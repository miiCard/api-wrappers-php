      Builds a new MiiCard object using the supplied OAuth credentials.

      __construct(string $consumerKey, string $consumerSecret, string $accessToken, string $accessTokenSecret) 

      A consumer key and secret are mandatory - without them an InvalidArgumentException is thrown. The caller may supply an access token and secret if they are known, or omit them if they intend to make requests that aren't signed by an access token and secret (for example, as would be the case during an initial OAuth exchange).

      makeSignedRequest(string $url, mixed $params, array $headers, bool $rawPostBody) 
      Inherited

      Makes signed requests both during the initial OAuth exchange (where OAuth parameters are sent form-encoded as part of the body of the request, with $rawPostBody = false) and afterwards when accessing the API (when JSON-encoded parameters are sent raw in the body of the request).

      inherited_from \OAuthSignedRequestMaker::makeSignedRequest()

      Parameters

      $url

      string

      The URL to be requested.

      $params

      mixed

      An array of parameters to be sent with the request as a post body (if $rawPostBody = false), or a string containing the raw post body to be sent (if $rawPostBody = true).

      $headers

      array

      Additional HTTP headers to be sent with the request.

      $rawPostBody

      bool

      If true, $params is interpreted as pre-parsed content to be dropped into the body of the request as-is. If false, the default, $params is interpreted as an array of key-value pairs to be form-encoded and sent in the body of the request.
